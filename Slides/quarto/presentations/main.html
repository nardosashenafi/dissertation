Hybrid Systems

  • exhibit both continuous state flow and discrete state transitions
  • E.g. Room temperature control, contact-rich robots
  • We tackle two main challenges in the control of contact-rich robots.

1. Mode Changes in Contact-Rich Robots



  • Contacts cause mode changes
  • Each mode has distinct dynamics
  • Controlled by switching amongst several policies

Motivation



  • Switching control does not scale well
    • E.g. Cooperative manipulation
    • kk contacts can have upto 2k2^k contact combinations
  • It is difficult to parameterize the switching condition


1. Proposed Method



  • Our method uses data-driven technique to automatically learn the mixture of expert controller and the switching conditionals
  • We design contact-aware controllers by training on trajectories generated from accurate contact models

Linear Complementarity Formulation

  • Suppose we have a system with kk potential contacts.
  • x=(q,q̇)x = (q, \dot{q}), where qq is position vector.
  • λN\lambda_N denotes normal contact forces, λT\lambda_T denotes tangential contact force (Coulomb friction)
  • Kinematic constraints:
  • gN∈ℝkg_N \in \mathbb{R}^k denotes normal distance between contact surfaces
  • γN(q,q̇)=∂gN∂t\gamma_N(q, \dot{q}) = \frac{\partial g_N}{\partial t} is the relative normal velocities
  • γN(q,q̇)=∂gN∂qq̇\gamma_N(q, \dot{q}) = \frac{\partial g_N}{\partial q} \dot{q} is the relative normal velocities
  • γN(q,q̇)=WNq̇\gamma_N(q, \dot{q}) = W_N \dot{q} is the relative normal velocities
  • γT(q,q̇)=WTq̇\gamma_T(q, \dot{q}) = W_T \dot{q} is the relative tangential velocities

Dynamic constraints

  • Model of contact dynamics: manipulator equations

M(q)q̈+h(q,q̇)−WNλN−WTλT=0,h(q,q̇)=C(q,q̇)q̇+G(q)−Bu(q,q̇),\begin{align*} \begin{gathered} M(q) \ddot{q} + h(q, \dot{q}) - W_N \lambda_N - W_T \lambda_T = 0, \\ h(q, \dot{q}) = C(q, \dot{q})\dot{q} + G(q) - Bu(q, \dot{q}), \end{gathered} \end{align*}

  • Notice q̈\ddot{q} is does not exist during impacts
  • (Moreau 1988) provides compact dynamical model which includes impact as M(q)ⅆq̇+h(q,q̇)ⅆt−ⅆR=0,ⅆR=WNλ̇Nⅆt+WTλ̇Tⅆt.\begin{align*} M(q) \dd \dot{q} + h(q, \dot{q}) \dd t - \dd R = 0, \\ \dd R = W_N \dot{\lambda}_N \dd t + W_T \dot{\lambda}_T \dd t. \end{align*}

Linear Complementarity Problem (LCP)


  • Objective: pose the complementarity formulation as quadratic function over the contact forces
  • Define λR:=μλN+λT,λL:=μλN−λT,\begin{align*} \lambda_R := \mu \lambda_N + \lambda_T, \\ \lambda_L := \mu \lambda_N - \lambda_T, \end{align*}
  • Corresponding complementarity is defined 0≤(ξR(q,q̇)ξL(q,q̇))⊥(λRλL)≥0,\begin{equation*} \begin{gathered} 0 \leq \begin{pmatrix} \xi_R(q, \dot{q}) \\ \xi_L(q, \dot{q}) \end{pmatrix} \perp \begin{pmatrix} \lambda_R \\ \lambda_L \end{pmatrix} \geq 0, \end{gathered} \end{equation*}

Background

Passive System Theory and Passivity-Based Control (PBC)

Dissipativity

A dynamical system

Σ:{ẋ=f(x,u)y=h(x,u)x∈𝒳⊂ℝ2n,u∈𝒰⊂ℝm \Sigma: \quad \begin{cases} \dot{x} &= f(x,u) \\ y &= h(x,u) \end{cases} \quad x \in \mathcal{X} \subset \mathbb{R}^{2n}, \, u \in \mathcal{U} \subset \mathbb{R}^{m}

is dissipative with respect to some supply rate ss if there exists a storage function ℋ:𝒳→ℝ+\mathcal{H}: \mathcal{X} \to \mathbb{R}^{+} such that

ℋ(x(t1))≤ℋ(x(t0))+∫t0t1s(u(t),y(t))dt \mathcal{H}\left( x(t_1) \right) \leq \mathcal{H}\left( x(t_0) \right) + \int_{t_0}^{t_1} s(u(t), y(t)) \, \text{d}t

for all x(t0)=x0x(t_0) = x_0, all input uu, and all t1≥t0t_1 \geq t_0

Dissipativity

ℋ(x(t1))≤ℋ(x(t0))+∫t0t1s(u(t),y(t))dt \mathcal{H}\left( x(t_1) \right) \leq \mathcal{H}\left( x(t_0) \right) + \int_{t_0}^{t_1} s(u(t), y(t)) \, \text{d}t

Dissipation Inequality

  • Stored energy at t1t_1 is at most equal to stored energy at t0t_0, plus externally supplied energy s(u,y)s(u,y)
  • No generation of energy, only internal dissipation
  • With s(t)≡0s(t) \equiv 0, trajectories tend towards minimum of ℋ\mathcal{H}

Passivity

The system Σ\Sigma is passive if it is dissipative with supply rate

s=u⊤y.s = u^\top y.

It is output strictly passive if it is dissipative with supply rate

s=u⊤y−δ∥y∥2,δ>0.s = u^\top y - \delta \lVert y \rVert^2, \; \delta > 0.

It is input strictly passive if it is dissipative with supply rate

s=u⊤y−δ∥u∥2,δ>0.s = u^\top y - \delta \lVert u \rVert^2, \; \delta > 0.

Stability of Passive Systems

Σ:{ẋ=f(x,u),f(0,0)=0,y=h(x,u),h(0,0)=0, \Sigma: \quad \begin{cases} \dot{x} &= f(x,u), && f(0,0) = 0, \\ y &= h(x,u), && h(0,0) = 0, \end{cases}

u⊤y≥ℋ̇=∂ℋ∂xf(x,u),ℋ≥0,y≡0⟹x≡0u^{\top} y \geq \dot{\mathcal{H}} = \frac{\partial\mathcal{H}}{\partial x} f(x,u),\quad \mathcal{H} \geq 0,\quad y \equiv 0 \implies x \equiv 0

Lemma (Khalil, 2002)

The origin of Σ\Sigma is:

  • stable if Σ\Sigma is passive,
  • asymptotically stable if Σ\Sigma is output strictly passive,
  • globally asymptotically stable if Σ\Sigma is output strictly passive and the storage function ℋ(x)→∞\mathcal{H}(x) \to \infty as ∥x∥→∞\lVert x \rVert \to \infty (radially unbounded)
  • Notice that we only require that the function ℋ\mathcal{H} be positive semidefinite and not necessarily positive definite as per usual Lyapunov conditions.
  • Observability (detectability) ensures LaSalle’s theorem works.
