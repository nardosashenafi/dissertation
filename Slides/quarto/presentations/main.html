Hybrid Systems

  • exhibit both continuous state flow and discrete state transitions
  • E.g. Room temperature control, contact-rich robots
  • We tackle two main challenges in the control of contact-rich robots.

1. Mode Changes in Contact-Rich Robots



  • Contacts cause mode changes
  • Each mode has distinct dynamics
  • Controlled by switching amongst several policies

Motivation



  • Switching control does not scale well
    • E.g. Cooperative manipulation
    • kk contacts can have upto 2k2^k contact combinations
  • It is difficult to parameterize the switching condition


1. Proposed Method



  • Our method uses data-driven technique to automatically learn a mixture of expert controller and the switching conditionals


  • We design contact-aware controllers by training on trajectories generated from accurate contact models
    • Contact-aware controllers minimize the adverse effects of contacts or leverage the potential contacts to their advantage

Contact Modeling

  • Objective: accurately model contacts, impacts and Coulomb friction.
  • E.g. Model of bouncing ball
  • Event detection searches for zero-crossings of the gap function
  • It fails during rapid contact

Linear Complementarity Formulation

  • Suppose we have a system with kk potential contacts.
  • x=(q,q̇)x = (q, \dot{q}), where q∈ℝmq \in \mathbb{R}^m is position vector.
  • λN∈ℝk\lambda_N \in \mathbb{R}^k denotes normal contact forces, λT∈ℝk\lambda_T \in \mathbb{R}^k denotes tangential contact force (Coulomb friction)
  • Geometric and kinematic constraints are defined as follows:
  • gN∈ℝkg_N \in \mathbb{R}^k denotes normal distance between contact surfaces
  • γN(q,q̇)=∂gN∂t\gamma_N(q, \dot{q}) = \frac{\partial g_N}{\partial t} is the relative normal velocities
  • γN(q,q̇)=∂gN∂qq̇\gamma_N(q, \dot{q}) = \frac{\partial g_N}{\partial q} \dot{q} is the relative normal velocities
  • γN(q,q̇)=WNq̇\gamma_N(q, \dot{q}) = W_N \dot{q} is the relative normal velocities
  • γT(q,q̇)=WTq̇\gamma_T(q, \dot{q}) = W_T \dot{q} is the relative tangential velocities

Dynamic constraints


  • Model of contact dynamics: manipulator equations

M(q)q̈+h(q,q̇)−WNλN−WTλT=0,h(q,q̇)=C(q,q̇)q̇+G(q)−Bu(q,q̇),\begin{align*} \begin{gathered} M(q) \ddot{q} + h(q, \dot{q}) - W_N \lambda_N - W_T \lambda_T = 0, \\ h(q, \dot{q}) = C(q, \dot{q})\dot{q} + G(q) - Bu(q, \dot{q}), \end{gathered} \end{align*}

  • Notice q̈\ddot{q} is does not exist during impacts

  • (Moreau 1988) provides compact dynamical model which includes impact as M(q)ⅆq̇+h(q,q̇)ⅆt−ⅆR=0,ⅆR=WNλ̇Nⅆt+WTλ̇Tⅆt.\begin{align*} M(q) \dd \dot{q} + h(q, \dot{q}) \dd t - \dd R = 0, \\ \dd R = W_N \dot{\lambda}_N \dd t + W_T \dot{\lambda}_T \dd t. \end{align*}

  • We integrate these equations via Moreau’s time stepping algorithm

Linear Complementarity Problem (LCP)


  • Objective: pose the complementarity formulation as quadratic function over the contact forces

  • Define λR:=μλN+λT,λL:=μλN−λT,\begin{align*} \lambda_R := \mu \lambda_N + \lambda_T, \\ \lambda_L := \mu \lambda_N - \lambda_T, \end{align*}

  • Corresponding complementarity is defined 0≤(ξR(q,q̇)ξL(q,q̇))⊥(λRλL)≥0,\begin{equation*} \begin{gathered} 0 \leq \begin{pmatrix} \xi_R(q, \dot{q}) \\ \xi_L(q, \dot{q}) \end{pmatrix} \perp \begin{pmatrix} \lambda_R \\ \lambda_L \end{pmatrix} \geq 0, \end{gathered} \end{equation*}

  • These definitions help express ξN,ξR,ξL\xi_N, \xi_R, \xi_L as affine functions of λN,λR,λL\lambda_N, \lambda_R, \lambda_L

(ξNξRλL)=A(λNλRξL)+b,\begin{align*} \begin{pmatrix} \xi_N \\ \xi_R \\ \lambda_L \end{pmatrix} = A \begin{pmatrix} \lambda_N \\ \lambda_R \\ \xi_L \end{pmatrix} + b, \end{align*}

LCP

  • We substitute the affine functions into the complementarity formulation

0≤(ξN(q,q̇)ξR(q,q̇)ξL(q,q̇))⊥(λNλRλL)≥0,\begin{equation*} \begin{gathered} 0 \leq \begin{pmatrix} \xi_N(q, \dot{q}) \\ \xi_R(q, \dot{q}) \\ \xi_L(q, \dot{q}) \end{pmatrix} \perp \begin{pmatrix} \lambda_N \\ \lambda_R \\ \lambda_L \end{pmatrix} \geq 0, \end{gathered} \end{equation*}

0≤[A(λNλRξL)+b]⊥(λNλRξL)≥0\begin{align*} 0 \leq \left[ A \begin{pmatrix} \lambda_N \\ \lambda_R \\ \xi_L \end{pmatrix} + b \right] \perp \begin{pmatrix} \lambda_N \\ \lambda_R \\ \xi_L \end{pmatrix} \geq 0 \end{align*}

  • The LCP can be posed as a feasibility problem and solved for λN,λR,ξL\lambda_N, \lambda_R, \xi_L
  • In the presence of friction, the LCP is a non-convex optimization problem
  • We use pivotting (basis-exchange) technique called Lemke’s algorithm to solve the LCP

Mixture of Experts (MOE)

  • typically used to fit ensemble of expert models to high variance or multi-modal dataset
  • This architecture provides the experts and a gating network, which divides the input space xx into state partitions within which a single expert is active

Mixture of Experts (MOE)


  • We select maximum number of experts
  • Let us denote the experts with F(x;θ)={F1(x;θ1),…,FNF(x;θNF)}F(x;\theta) = \{F_1(x;\theta_1), \dots , F_{N_F}(x;\theta_{N_F}) \}, and their parameters are given by the set θ={θ1,…,θNF}\theta = \{\theta_1, \dots , \theta_{N_F} \}
  • Each expert FiF_i resides in a particular state partition ii
  • The gating network 𝐏(x|ψ)\mathbf{P}(x| \psi) provides the probability of input state xx belonging to each state partition. It is denoted by

𝐏(x|ψ):=(P1(x|ψ),…,PNF(x|ψ))\begin{align*} \mathbf{P}(x| \psi) := (P_1(x| \psi), \dots, P_{N_F}(x | \psi)) \end{align*}

  • The objective is to learn the parameters (ψ,θ)(\psi, \theta) that best fit the dataset

E.g. Old Faithful

  • Objective: learn two expert Gaussian models to predict duration given waiting time
  • The experts are chosen as F(x;θ)={𝒩1(μ1,σ1),𝒩2(μ2,σ2)} F(x;\theta) = \{\mathcal{N}_1(\mu_1, \sigma_1), \mathcal{N}_2(\mu_2, \sigma_2)\}
  • The gating network is a neural net 𝐏(x|ψ)=[P1(x|ψ),P2(x|ψ)]\mathbf{P}(x | \psi) = [P_1(x | \psi), P_2(x | \psi) ]
  • Learn the parameters θ={(μ1,σ1),(μ2,σ2)}\theta = \{(\mu_1, \sigma_1), (\mu_2, \sigma_2) \} and ψ\psi from dataset 𝔻={(x1,y1),…,(xN,yN)}\mathbb{D} = \{(x_1, y_1), \dots, (x_N, y_N) \}

Training


  • Training procedure:
    • Start from initial parameters (ψ,θ)(\psi, \theta)
    • Sample initial state x0x_0
    • Generate trajectories ϕ(x0,u,T)\phi(x_0, u, T) using current parameters i∼Categorical(𝐏(x|ψ))u(x;ψ,θ)=Fi(x;θi)M(q)ⅆq̇+h(x;ψ,θ)ⅆt−ⅆR=0 (Moreau’s time stepping)\begin{gather*} i \sim \text{Categorical} (\mathbf{P}(x| \psi)) \\ u(x; \psi, \theta) = F_i(x; \theta_i) \\ M(q) \dd \dot{q} + h(x; \psi, \theta) \dd t - \dd R = 0 \text{ (Moreau's time stepping)} \end{gather*}
    • Assign a running cost ℓ\ell to the trajectories based on performance
    • Update parameters (ψ,θ)(\psi, \theta) to minimize running cost

  1. Minimum Loss Trajectory (MTL):
  • Accumulated loss may not reflect desired behavior. E.g. Simple pendulum
  • Simple pendulum needs to pump slowly, which would accumulate large cost
  • MTL encourages trajectories to eventually lead to a minimum cost

Cartpole with wall barriers

  • Cartpole: pendulum hinged on an actuated cart
  • Objective: swingup the pendulum to the upright using the control authority on the cart
  • Linear Quadratic Regulator (LQR) IS used to stabilize after swingup

(Sirichotiyakul ACC 2022)

  • The wall barriers create a contact-rich (hybrid) system
  • Notice LQR fails due to the impact from the wall
  • We employ contact-aware MOE controller to
    • leverage the impacts from the wall
    • switch to a policy that assists in the catching process

Observations

  • The MOE controller switches just after impact
  • After impact, the active expert applies rapid braking

  • There is a relationship between the minimum gap between the pendulum and the walls, and the expert used
  • An expert is dedicated to states with low chance of contact and viceversa
