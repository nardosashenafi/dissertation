Classical Control vs Data-Driven

 

Classical Control

  • Derived from nominal model
  • System specific
  • Allows for stability analysis

 

Data-Driven

  • Finds controllers from repeated interactions
  • Flexible across systems
  • Lacks stability certificates

Motivation

 

There are advantages to merging machine learning with classical control theory

  • Use the insight from classical control theory to create robust data-driven controllers
    • Insights about stability and convergence
  • Solve difficulties in classical control via machine learning
